Cheatsheet CSS

CSS (ascading Style Sheet) est un language informatique qui permet de rajouter la couche présentation d'une page web. En effet, une page web est constituée de 3 niveaux:

  1. La structure assumée par html
  2. La présentation (style) rôle du CSS
  3. Le comportementendosé par javascript

Les 3 modalités visant à inclure du CSS dans une page web:

  1. Fichier externe: "style.css" dans l'en-tête de la page html entre les balises head: <link href= "style.css" rel="stylesheet" type="text/css">
  2. Fichier interne, déclare les styles, entre les balises head,qui ne s'appliqueront qu'à la page considérée
  3. Styles CSS au sein même du code html, inline, à l'intérieur des balises pour les éléments
La première est la plus aboutie car elle permet un travail de développement optimal car elle permet plus de flexibilité:
  • Application du même style à plusieurs pages différentes
  • Apport de nouvelles variantes stylistiques grace à un 3ème fichier lié à la page via la première ligne du fichier css sousla forme @ import "import style.css"
  • On n'est plus obligé de passer par le code html de chacune des pages pour apporter des rectifications
